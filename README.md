## LeetSave Chrome Extension

**LeetSave** is a simple, fun, and powerful Chrome extension for competitive programmers, students, and recruiters. Easily save and manage LeetCode profiles with just one click!

---

### üöÄ Features

- **One-Click Profile Saving:**  
  Save any LeetCode profile instantly from their profile page.

- **Smart Profile List:**  
  Search, filter, and manage all your saved profiles in a clean popup interface.

- **Platform Switch:**  
  View and manage your saved LeetCode profiles.

- **Export & Import:**  
  Backup or share your saved profiles with easy JSON export/import.

- **Quick Profile Access:**  
  Open any saved profile directly from the extension.

- **Minimalistic UI:**  
  Modern, responsive design with smooth animations for a delightful user experience.

### üì¶ Installation

1. Clone or download this repo.
2. Go to `chrome://extensions/` and enable **Developer Mode**.
3. Click **Load unpacked** and select the project folder.
4. Visit any LeetCode profile page and start saving!
